## Run Locally

Clone the project

```bash
  git 
```

Go to the project directory

```bash
  cd crypto_verse_2.0
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```


## Tech Stack

**Client:** React, Recoil, TailwindCSS, Framer Motion

**Server:** Rapid API Services


## Features

- Crypto Real Time Data
- Live previews of Crypto News
- Exchange Rates
- Cross platform
